DNAICI: Differential Network Analysis in Intra-chromosomal Community Interaction

DNAICI is a Python package for Differential Network Analysis in Intra-chromosomal Community Interaction

The README file provides instructions for analyzing demo data. The default values for parameters (e.g. cohort = ‘untreated’, chromosome = [‘chr18’,’chr19’]) are designed for demo data. For more details about demo data, please refer to [README.md].
